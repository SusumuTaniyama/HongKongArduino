説明

74シリーズはHCを使用したほうがよいです。がなくて交わっている線は結線しないという意味です。
74138のY0はプリンタポートのノイズにより不安定になるので使うのを避けました。(これでかなりはまった。)

回路記号表

記号 名称 説明
集合抵抗 片側が一つに結線された抵抗の集まり。共通の線には印がついている。
LED(発光ダイオード) 矢印の方向にしか電流を流さず、電流が流れると発行する素子。
電解コンデンサ 極性のあるコンデンサ。+側の端子の電圧が高くないといけない。
電源(Vcc) この記号のあるところはすべて結線し電源につなぐ。
GND(グランド） 接地。この記号のあるところはすべて結線する。電位がゼロであることを表す。
